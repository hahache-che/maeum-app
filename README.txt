📱 마음이 PWA 설치 방법
========================

📁 파일 구성
- index.html     (앱 메인 파일)
- manifest.json  (PWA 설정)
- sw.js          (오프라인 캐시)
- icon-192.svg   (앱 아이콘)
- icon-512.svg   (앱 아이콘 큰 것)
- README.txt     (이 파일)

🚀 설치 방법 (무료 호스팅 추천: GitHub Pages)

1. GitHub.com 가입/로그인
2. 새 Repository 만들기 (예: maeum-app)
3. 파일 5개 모두 업로드
4. Settings → Pages → Branch: main 선택 → Save
5. https://[내아이디].github.io/maeum-app/ 접속

📱 핸드폰에 앱으로 설치

[아이폰]
1. Safari로 위 주소 접속
2. 하단 공유 버튼(네모+화살표) 탭
3. "홈 화면에 추가" 선택

[안드로이드]
1. Chrome으로 위 주소 접속
2. 주소창 오른쪽 메뉴(점 3개) 탭
3. "앱 설치" 또는 "홈 화면에 추가" 선택

✨ 그러면 앱 아이콘이 홈 화면에 생겨요!
